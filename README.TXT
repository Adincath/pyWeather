PyWeather (Python3)
Brooks Becton

Initial Install:
    Unix:
        virtualenv env
        source env/bin/activate
        pip install -r requirements.txt
    Windows:
        env\scripts\activate.bat
        pip install -r requirements.txt

Commands:
    Specific Location:
        python pyWeather.py (location) (mode) (days in forecast)
    Default Location:
        python pyWeather.py 

Extras:
    virtualenv and python must be installed.
